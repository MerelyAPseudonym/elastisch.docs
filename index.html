Elastisch

A minimalistic Clojure client for ElasticSearch.

Minimalistic

Elastisch follows ElasticSearch REST API and does not introduce any new abstractions or terminology

Clojuric

With Elastisch, you work with Clojure data structures. Elastisch is pure Clojure. If Java interop is not your piece of cake, you'll feel at home

(Mostly) Feature Complete

Elastisch is not entirely feature complete but tries to be. Minimalitisc does not mean primitive: ElasticSearch has many powerful features users should benefit from.
